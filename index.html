Quarto Titulo

Quinto Titulo

Para estilizar o texto em italico usar a tag i, sublinhado usar a tag u, destaque usar a tag strong

Para colocar uma letra em baixoa usar a tag sub

Para colocar uma letra em cimaa usar a tag sup

Para usar fonte diferente usar a tag style e font, a tag font já não é mais usada

Deletar um algo e inserir algo no lugar usar as tags del e ins

Abv Abreviar usar tag abbr

Para colocar uma citação em bloco separado usar a tag blockquote
